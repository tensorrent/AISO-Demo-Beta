Any agent can drive this engine through two tools. In-page, call
        window.aiso.reason(query) and window.aiso.remember(content) from the console —
        each returns the exact JSON shown here. The companion machine endpoint
        (aiso-demo-server.mjs) exposes the same contract over HTTP
        (POST /v1/reason, POST /v1/remember, GET /v1/schema) for a headless agent.
